# Nu Kenzie
*customer-base*
Este é um aplicativo no qual você pode gerenciar seus contatos de maneira simples e fácil.

Rodando localmente
Clone o projeto

```shel
  git@github.com:Victor-Novakoski/react-nu-kenzie-victor.git
```  


Entre no diretório do projeto
```shel
  cd nu_kenzie
```
  ou
  
```shel
  code .
```


Instale as dependências
```shel
  yarn
```  


Inicie o servidor
```shel
  yarn dev
```  


Stack utilizada
Front-end: React, Styled Components e TypeScript
